 Aplikasi Kasir Bebrbasis GUI Java
 Proyek ini merupakan modul Halaman Login untuk Aplikasi Kasir (Point of Sale) berbasis Desktop yang dibangun menggunakan bahasa pemrograman Java dan library Swing/AWT. Fokus utama dari modul ini adalah memberikan gerbang masuk yang aman dan user-friendly bagi staf kasir sebelum mengakses fungsi utama transaksi.
Desain antarmuka menggunakan pendekatan Dual-Panel Layout di mana sisi kiri berfungsi sebagai elemen visual branding (grafik pertumbuhan bisnis) dan sisi kanan berfungsi sebagai kontrol panel autentikasi pengguna.

Ini adalah modul utama dari Sistem Informasi Kasir Toko Abadi, sebuah aplikasi desktop berbasis Java Swing yang dirancang untuk mempercepat proses transaksi ritel secara real-time. Halaman ini berfungsi sebagai pusat operasional di mana kasir dapat mengelola daftar belanja pelanggan, menghitung total biaya, hingga memproses pembayaran dan mencetak struk secara otomatis.

Aplikasi ini mengedepankan efisiensi kerja dengan menempatkan seluruh elemen krusial—mulai dari input data barang hingga modul pembayaran—dalam satu tampilan yang terintegrasi (Single Page Interface).
